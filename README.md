# Агент для проведения интервью
    Мультагентная система представляет собой набор из 4 агентов. 
    * Mentor
    * Interviewer
    * Vibemaster
    * Manager 

## Mentor 
    Оценивает фактический ответ пользователя при помощи набора метрик. Его вывод всегда структуририрован и он дает информацию агенту Interviewer. Формирует сущности вопросов

## Interviewer 
    Проводит собеседование с кандидатом. Задает вопросы и отвечает на его вопросы. Пытается не уходить от темы. 

## Vibemaster
    Просто оценивает настроение кандидата. Изначально планировалось, что будет оценивать софт скиллы кандидата, но этой логики нет. Зато отлично выбрасывает кандидатов из собеседования при первых признаках агрессии.

## Manager
    В конце графа всегда получает полный контекст сообщений пользователя и делает вывод исходя из оценок по каждому из вопросов


# Как поднять
    1. Клонируйте репозиторий
    2. Возьмите у мистрали API ключ
    3. Создайте .env файл и укажите в нем MISTRAL_KEY=XXXXXX
    4. Запустите make build, затем make run
